# CCUsage Resident Monitor

macOSのメニューバーに常駐するClaude Code使用量監視ツール

## 概要

ccusageを使用してClaude Codeの使用量をリアルタイムで監視し、メニューバーに表示するmacOSネイティブアプリケーションです。

## 機能

- メニューバーでの使用量表示
- リアルタイム監視
- 使用量警告
- プログレス表示
- 設定可能な閾値

## 技術スタック

- Swift
- SwiftUI
- MenuBarExtra
- ccusage CLI integration
